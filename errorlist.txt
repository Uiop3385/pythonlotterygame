List of all errors ([E-0X] or [E-0Xx]):
  E-01 : 
    Cause : Happens when a starting day value superior to 365 is entered in debug mode.
    Fix : Enter a valid value, that is not higher than 365.
  E-02 :
    Cause : Happens when a moners value superior to 999999999999999 is entered in debug mode.
    Fix : Enter a valid value, that remains below 999999999999999.
  E-03a :
    Cause : Happens when an invalid choice is entered on the main menu.
    Fix : Enter a valid choice.
  E-04 :
    Cause : Happens when a starting day value inferior to 1 is entered in debug mode.
    Fix : Enter a valid value that is superior to 0.
  E-05 :
    Cause : Happens when a negative moners value is entered in debug mode.
    Fix : Enter a positive moners value.
  E-06 :
    Cause : Happens when a decimal moners value is entered in debug mode.
    Fix : Enter an integer value.
  E-07 :
    Cause : Happens when a decimal day is entered in debug mode.
    Fix : Enter an integer value.
